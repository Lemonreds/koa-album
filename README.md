## koa-album

a web album application based on koa and node.

预览 [Live Preview](http://119.29.128.237:8888/) 

## Local Preview

install dependencies
```
npm install 
```

start service
```
node app.js
```
server will start at localhost: 8888.


modification 
```
gulp dev
```

minify static source 
```
gulp build
```
