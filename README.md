## koa-album

a web album application based on koa and node. 
一个基于Koa2的Node的web相册应用。

预览 [Live Preview](http://119.29.128.237) 

## Local Preview

install dependencies / 安装依赖包
```
npm install 
```

start service  / 启动服务 
```
node app.js
```
server will start at localhost: 8888. / 本地服务


modification  / 开发环境 
```
gulp dev
```

minify static source / 压缩静态文件
```
gulp build
```
